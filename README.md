Api created using php and SlimFramework


Installation 
- import _sql/slimapp.sql
- edit database entries in db.php
- install slim with composer 


API can do
 - GET /api/customers
 - GET /api/customer/{id}
 - POST /api/customer/add
 - PUT /api/customer/update/{id}
 - DELETE /api/customer/delete/{id}
